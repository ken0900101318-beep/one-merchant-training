1. 系統簡介

🎯 ONE桌遊商家端是什麼？

系統定位

ONE桌遊 24H 自助共享管理系統

核心功能：提供商家完整的店務管理、會員管理、設備控制、金流設定等一站式解決方案

📱 系統特色

  • ✅ 24小時自助營運：支援無人化自助開桌模式
  • ✅ 智能設備整合：IoT 智能門鎖、電源控制、藍牙喇叭等
  • ✅ 多店管理：一個帳號可管理多家分店

  • ✅ 金流整合：藍新金流、LINE PAY、Apple Pay

3. 店家管理

📍 功能位置

路徑：我的 → 店家管理

🎯 功能說明

店家管理是設定店家基本資訊的地方，包含地址、電話、營業時間、封面照片等，這些資訊會顯示在用戶端 APP 的店家頁面。

⚠️ 重要提醒

店家資訊會直接影響顧客的第一印象，請務必填寫完整且正確！

📝 必填欄位

欄位名稱 說明 範例
店家名稱 顯示在 APP 上的店名 ONE桌遊-台北信義店
店家地址 完整地址（含縣市、區、路街號） 台北市信義區信義路五段7號
聯絡電話 顧客可撥打的客服電話 02-1234-5678
營業時間 24 小時制，可設定不同時段 00:00 - 24:00（24小時營業）
店家封面照片 用戶端首頁顯示的橫幅照片 建議尺寸 1200×600 px
繳費機照片 店內繳費機位置的照片（方便顧客找到） 建議拍攝繳費機與周邊環境

🖼️ 照片上傳建議

  • ✅ 店家封面：展現店內環境、包廂氛圍
  • ✅ 繳費機照片：清楚標示位置，最好包含明顯地標
  • ✅ 照片大小：建議 500KB 以內（系統會自動壓縮）
  • ✅ 照片格式：JPG、PNG 皆可

⚙️ 操作步驟

  1. 點選底部導航列「我的」
  2. 點選「店家管理」
  3. 填寫店家基本資訊（名稱、地址、電話、營業時間）
  4. 上傳店家封面照片
  5. 上傳繳費機照片
  6. 點擊「保存」按鈕

✅ 完成確認

設定完成後，請至用戶端 APP 檢查店家資訊是否正確顯示。

4. 套票設定

📍 功能位置

路徑：我的 → 套票設定

🎯 功能說明

套票是顧客預約包廂時可選擇的時段方案，例如「2小時 $200」、「4小時 $350」等。

套票 ≠ 包廂，套票是時長與價格的組合，可綁定到多個包廂。

📋 套票範例

套票名稱 時長 價格 適用場景
快閃方案 1 小時 $120 短暫聚會、試玩
標準方案 2 小時 $200 一般打牌時長
歡樂方案 4 小時 $350 長時間聚會
通宵方案 8 小時 $600 深夜至清晨

🔧 套票設定項目

必填欄位

  • 套票名稱：顯示在用戶端的名稱（例：歡樂4小時）
  • 時長：以小時為單位（支援 1-24 小時）
  • 價格：新台幣（NT$）
  • 是否啟用：關閉後，用戶端不會顯示此套票

選填欄位

  • 優惠價：設定促銷價格（會顯示原價刪除線）
  • 適用時段：例如僅限平日、僅限假日等
  • 套票說明：補充說明（例：含飲料一杯）

⚙️ 操作步驟

  1. 點選「我的」→「套票設定」
  2. 點擊「新增套票」按鈕
  3. 填寫套票名稱、時長、價格
  4. 設定是否啟用（預設為啟用）
  5. （選填）設定優惠價或適用時段
  6. 點擊「保存」

💡 套票規劃建議

  • 建議至少設定 3-5 種套票（涵蓋短、中、長時段）
  • 價格設定可參考「每小時 $80-120」的定價
  • 長時段套票可設定優惠價，吸引長時間消費
  • 假日/平日可設定不同價格（需使用「適用時段」功能）

⚠️ 注意事項

  • 套票設定後，需至「包廂管理」將套票綁定到包廂
  • 已綁定包廂的套票無法刪除（僅能停用）
  • 修改套票價格不會影響已成立的訂單

5. 包廂管理

📍 功能位置

路徑：我的 → 包廂管理

🎯 功能說明

包廂管理用於設定每個包廂的名稱、可綁定套票、照片、設備資訊等。

包廂通常由總部協助建立（依據現場裝潢平面圖），店家主要負責綁定套票與上傳照片。

📋 包廂資訊欄位

欄位名稱 說明 編輯權限
包廂編號 系統自動編號（例：101、102...） 🔒 總部設定
包廂名稱 顯示名稱（例：風雲再起、天龍八部） ✅ 可編輯
包廂照片 包廂內部照片（用戶端顯示） ✅ 可編輯
可選套票 勾選此包廂可使用的套票 ✅ 可編輯
設備清單 門鎖、電源、喇叭等 IoT 設備綁定 🔒 總部設定
是否啟用 關閉後，用戶端無法預約此包廂 ✅ 可編輯

⚙️ 操作步驟：綁定套票

  1. 點選「我的」→「包廂管理」
  2. 選擇要編輯的包廂
  3. 勾選「可選套票」（可多選）
  4. 點擊「保存」

📷 上傳包廂照片

  1. 進入包廂編輯頁面
  2. 點擊「上傳照片」按鈕
  3. 選擇照片（建議 800×600 px 以上）
  4. 點擊「保存」

💡 照片拍攝建議

  • ✅ 光線充足：避免陰暗或曝光不足
  • ✅ 整潔乾淨：拍攝前整理包廂
  • ✅ 展現特色：拍攝麻將桌、沙發、裝潢等亮點
  • ✅ 角度選擇：建議從門口往內拍，展現整體空間

🔧 啟用 / 停用包廂

何時需要停用包廂？

  • 🔧 設備維修中（門鎖故障、冷氣故障）
  • 🚧 包廂整修施工
  • 🧹 深度清潔暫停開放

停用步驟

  1. 進入包廂編輯頁面
  2. 將「是否啟用」設為關閉
  3. 點擊「保存」

⚠️ 注意事項

  • 停用包廂後，已預約的訂單不受影響（依然有效）
  • 新顧客無法預約已停用的包廂
  • 若包廂有進行中訂單，建議待訂單結束後再停用

6. 權限與店員管理

📍 功能位置

路徑：我的 → 權限管理 / 店員列表

🎯 功能說明

權限管理用於設定不同角色的操作權限，店員列表用於新增員工帳號並分配角色。

👥 預設角色與權限

角色名稱 權限範圍 適用對象
店主 🔓 所有功能（含金流設定、權限管理） 店家負責人
店長 ✅ 訂單管理、會員管理、包廂管理
❌ 金流設定、權限管理		 店長、主管
店員 ✅ 訂單查詢、會員查詢
❌ 無法修改設定		 一般店員
清潔人員 ✅ 查看包廂狀態、標記清潔完成
❌ 無法查看訂單金額		 清潔人員
客服 ✅ 會員管理、訂單處理、發放卡券
❌ 無法修改店家設定		 客服人員

🔧 新增自訂角色

  1. 點選「我的」→「權限管理」
  2. 點擊「新增角色」
  3. 輸入角色名稱（例：夜班主管）
  4. 勾選允許的功能權限
  5. 點擊「保存」

👤 新增店員帳號

  1. 點選「我的」→「店員列表」
  2. 點擊「新增店員」
  3. 填寫店員資料：
    • 姓名
    • 手機號碼（作為登入帳號）
    • 選擇角色（店長、店員、清潔人員等）
  4. 點擊「保存」
  5. 系統會發送簡訊通知店員（含初始密碼）

💡 初始密碼說明

新增店員後，系統會自動產生隨機密碼並透過簡訊發送。

店員首次登入後，必須修改密碼才能繼續使用。

🔄 修改店員角色

  1. 進入「店員列表」
  2. 點選要修改的店員
  3. 更改「角色」欄位
  4. 點擊「保存」

🗑️ 停用 / 刪除店員

停用 vs 刪除

操作 效果 適用場景
停用 帳號保留，但無法登入 員工暫時離職、休假
刪除 帳號永久移除 員工已離職且確認不再回任

⚠️ 安全建議

  • 🔐 店主帳號不應分享給他人
  • 🔐 離職員工請立即停用或刪除帳號
  • 🔐 定期檢查「店員列表」，確認無多餘帳號

7. 優惠券與電子票券

📍 功能位置

路徑：我的 → 優惠券設定 / 電子票券設定

🎯 兩者差異

類型 功能 範例
優惠券 結帳時折扣（滿額折、折扣碼） 滿 $500 折 $50
電子票券 可兌換實際商品或服務 續時券（兌換1小時）、免費券

🎫 優惠券設定

優惠券類型

  • 滿額折：消費滿 $X 折 $Y（例：滿500折50）
  • 折扣碼：輸入代碼享優惠（例：NEW100 折 $100）
  • 百分比折扣：全單打 X 折（例：8折券）

設定步驟

  1. 點選「我的」→「優惠券設定」
  2. 點擊「新增優惠券」
  3. 填寫優惠券資訊：
    • 優惠券名稱（例：新客首購優惠）
    • 優惠類型（滿額折、折扣碼、百分比）
    • 折扣金額或折扣百分比
    • 有效期限
    • 使用條件（最低消費金額）
  4. 點擊「保存」

🎟️ 電子票券設定

票券類型範例

票券名稱 兌換內容 適用場景
續時券 1H 延長1小時 活動贈送、會員福利
續時券 2H 延長2小時 VIP會員專屬
免費體驗券 免費使用2小時 新客推廣、試營運
飲料兌換券 兌換指定飲料一杯 消費滿額贈送

設定步驟

  1. 點選「我的」→「電子票券設定」
  2. 點擊「新增票券」
  3. 填寫票券資訊：
    • 票券名稱（例：續時券1H）
    • 票券說明（兌換內容）
    • 有效期限
    • 是否可與優惠券同時使用
  4. 點擊「保存」

🎁 如何發放券給會員？

方法一：手動發放（個別會員）

  1. 進入「會員管理」
  2. 搜尋並點選會員
  3. 點擊「發放卡券」
  4. 選擇要發放的優惠券或電子票券
  5. 填寫數量
  6. 點擊「確認發放」

方法二:自動發放（運營活動）

透過「運營活動」設定觸發條件（例：首次消費送續時券），詳見本文件第 9 章。

💡 優惠策略建議

  • ✅ 新客首購：發放折扣券吸引首次消費
  • ✅ 生日禮：會員生日月發放續時券
  • ✅ 滿額贈：消費滿 $1,000 送續時券 1H
  • ✅ 好友推薦：推薦新會員各得 $100 折扣券

⚠️ 注意事項

  • 已發放的券無法收回
  • 券過期後自動失效，會員無法使用
  • 建議定期檢查券的使用率，調整發放策略

8. 儲值優惠設定

📍 功能位置

路徑：我的 → 儲值優惠

🎯 功能說明

儲值優惠是鼓勵會員預先儲值的促銷方案，例如「儲值 $1,000 送 $100」。

會員儲值後，餘額存於本店帳戶，僅能在本店使用（不可跨店）。

💰 儲值方案範例

方案名稱 儲值金額 贈送金額 實際可用
小額儲值 $500 +$30 $530
標準儲值 $1,000 +$100 $1,100
超值儲值 $2,000 +$300 $2,300
VIP 儲值 $5,000 +$1,000 $6,000

⚙️ 設定步驟

  1. 點選「我的」→「儲值優惠」
  2. 點擊「新增方案」
  3. 填寫方案資訊：
    • 方案名稱（例：超值儲值）
    • 儲值金額（會員實際支付）
    • 贈送金額（額外贈送）
    • 是否啟用
  4. 點擊「保存」

📊 餘額查詢與管理

會員如何查詢餘額？

  • 💵 提升客單價：會員有餘額時消費意願更高

10. 金流設定

📍 功能位置

路徑：我的 → 支付配置

🎯 功能說明

金流設定用於串接第三方支付服務，讓顧客可透過信用卡、LINE PAY、Apple Pay 等方式線上付款。

ONE桌遊系統支援：

  • ✅ 藍新金流（Neweb Pay）
  • ✅ LINE PAY

⚠️ 重要提醒

金流申請需要公司營業登記，個人無法申請。

申請審核時間約 5-7 個工作天，請提前準備。

💳 支付方式比較

金流服務 手續費 支援付款方式 適用對象
藍新金流 2.8% - 3.2% 信用卡、ATM、超商代碼 所有店家
LINE PAY 2.5% - 3.0% LINE PAY、信用卡 重視 LINE 生態圈的店家

💡 建議

建議同時申請藍新金流與 LINE PAY，提供顧客更多付款選擇。

10-A. 藍新金流申請

📋 申請前準備

  • ✅ 公司營業登記證明
  • ✅ 負責人身分證
  • ✅ 公司銀行帳戶（撥款用）
  • ✅ 公司大小章

⚙️ 申請步驟

  1. 前往藍新金流官網：https://www.newebpay.com/
  2. 點擊「商店申請」
  3. 填寫基本資料（公司名稱、統編、地址、負責人）
  4. 上傳證明文件（營業登記證、身分證）
  5. 設定收款銀行帳戶
  6. 提交申請並等待審核（約 5-7 個工作天）
  7. 審核通過後，藍新金流會寄送商店代號與HashKey / HashIV

🔧 將藍新金流串接到 ONE桌遊系統

  1. 取得藍新金流提供的：
    • 商店代號（MerchantID）
    • HashKey
    • HashIV
  2. 聯繫ONE桌遊總部提供上述資料
  3. 總部協助完成系統串接設定
  4. 測試付款流程（小額測試訂單）
  5. 確認無誤後正式啟用

✅ 設定完成確認

設定完成後，請至用戶端 APP 測試下單，確認可正常跳轉至付款頁面。

10-B. LINE PAY 申請

📋 申請前準備

  • ✅ 公司營業登記證明
  • ✅ 負責人身分證
  • ✅ 公司銀行帳戶
  • ✅ LINE 官方帳號（建議先申請）

⚙️ 申請步驟

  1. 前往LINE PAY 商家中心：https://pay.line.me/tw/merchants
  2. 點擊「立即申請」
  3. 填寫商家資料（公司名稱、統編、營業項目）
  4. 上傳證明文件
  5. 設定收款帳戶
  6. 提交申請並等待審核（約 5-7 個工作天）
  7. 審核通過後，LINE PAY 會提供Channel ID 與 Channel Secret Key

🔧 將 LINE PAY 串接到 ONE桌遊系統

  1. 取得 LINE PAY 提供的：
    • Channel ID
    • Channel Secret Key
  2. 聯繫ONE桌遊總部提供上述資料
  3. 總部協助完成系統串接設定
  4. 測試付款流程
  5. 確認無誤後正式啟用

💡 LINE PAY 的優勢

  • ✅ 台灣用戶普及率高（超過 80% 使用 LINE）
  • ✅ 付款流程簡單（LINE APP 內完成）
  • ✅ 可整合 LINE 官方帳號行銷

11. 預約訂單設置

📍 功能位置

路徑：我的 → 預約訂單設置

🎯 功能說明

預約訂單設置用於設定預約規則、取消政策、下單須知等，這些規則會顯示在用戶端 APP 的預約頁面。

📋 設定項目

設定項目 說明 建議設定
可預約天數 最多可預約幾天後的包廂 30-60 天
最短預約時長 最少需訂幾小時 1 小時
最長預約時長 最多可訂幾小時 8-12 小時
取消訂單規則 多久前可免費取消 開始前 2 小時
逾時付款規則 下單後多久未付款自動取消 10 分鐘
下單須知 顧客下單前需同意的注意事項 自訂文字說明

🔧 取消訂單規則詳解

範例設定

  • 規則 1：開始前 2 小時以上取消 → 全額退款
  • 規則 2：開始前 2 小時內取消 → 不退款（金額轉為本店餘額）
  • 規則 3：訂單已開始 → 無法取消

為什麼需要取消規則？

避免顧客臨時取消造成包廂空置，影響店家營收。

📝 下單須知範例

建議內容：

  • ✅ 請準時到場，逾時不候
  • ✅ 包廂內禁止吸菸、飲食（若店內有規定）
  • ✅ 請愛護設備，損壞需照價賠償
  • ✅ 訂單開始前 2 小時內取消，不予退款
  • ✅ 如需延長時間，請至 APP 操作續時

⚙️ 操作步驟

  1. 點選「我的」→「預約訂單設置」
  2. 填寫各項設定：
    • 可預約天數
    • 最短/最長預約時長
    • 取消訂單規則
    • 逾時付款規則
  3. 編輯「下單須知」內容
  4. 點擊「保存」

⚠️ 法律提醒

根據消費者保護法，若顧客主動取消訂單且符合退款條件，店家必須退款。

建議明確告知取消規則，避免糾紛。

  • ✅ 定期保養：每月清潔投幣口與找零機構
  • ✅ 備用硬幣：準備充足的 $1、$5、$10 硬幣

⚠️ 常見問題

  • ❓ 繳費機不找零 → 檢查硬幣盒是否已滿或硬幣不足
  • ❓ 無法投幣 → 清潔投幣口，檢查是否有卡幣
  • ❓ 金額顯示錯誤 → 重新啟動繳費機

📋 訂單狀態說明

訂單狀態 說明
🔵 進行中 顧客已入場，訂單尚未結束
🟡 待付款 顧客已下單但尚未付款（10分鐘內需付款）
🟢 已完成 訂單已結束，款項已入帳
🟠 已取消 顧客取消訂單或系統自動取消

  4. （選填）選擇日期範圍
  5. 查看訂單列表

點擊訂單可查看：

  • ✅ 訂單編號

  4. 確認加價金額
  5. 點擊「確認延長」
  6. 顧客會收到延長通知並需線上付款

  2. 點擊「申請退款」

  4. 填寫退款金額

⚠️ 注意事項

  • 延長訂單需顧客同意並付款後才生效
  • 提早結束訂單，系統會自動計算退款金額
  • 退款需總部審核，請耐心等待

22. 常見問題處理（FAQ）

🔧 系統操作問題

Q1: 忘記商家端登入密碼怎麼辦？

A:

  1. 點擊登入頁面的「忘記密碼」
  2. 輸入註冊的手機號碼
  3. 系統會發送驗證碼簡訊
  4. 輸入驗證碼後設定新密碼

Q2: 商家端 APP 無法登入？

A: 請確認以下事項：

  • ✅ 帳號密碼是否正確
  • ✅ 網路連線是否正常
  • ✅ APP 版本是否為最新（至 App Store / Google Play 更新）
  • ✅ 若仍無法登入，請聯繫 ONE桌遊總部

Q3: 如何新增包廂？

A: 包廂新增需由ONE桌遊總部協助處理（涉及設備綁定與平面圖配置），請聯繫總部客服。

💳 金流與付款問題

Q4: 顧客反映付款失敗怎麼辦？

A: 請先確認：

  1. 顧客的付款方式是否正常（信用卡額度、LINE PAY 餘額）
  2. 檢查店家的金流設定是否正確（藍新金流或 LINE PAY）
  3. 請顧客重新嘗試付款
  4. 若持續失敗，請聯繫 ONE桌遊總部技術支援

Q5: 如何查詢營收撥款時間？

A:

  • 藍新金流：通常為 T+2 個工作天（例：週一交易，週三撥款）
  • LINE PAY：通常為 T+3 個工作天
  • 可登入金流商後台查詢撥款明細

🔒 設備與門鎖問題

Q6: 門鎖無法開啟怎麼辦？

A: 請依序檢查：

  1. 電量：檢查門鎖電池是否充足（設備管理 可查看電量）
  2. 網路：確認閘道器與路由器連線正常
  3. 重新啟動：至「設備管理」手動遠端開啟
  4. 備用方式：使用實體鑰匙開門（緊急用）
  5. 若仍無法開啟，請聯繫 ONE桌遊總部

Q7: 包廂電源無法控制？

A:

  • 檢查電源控制器是否離線（設備管理查看）
  • 重啟閘道器
  • 檢查 Wi-Fi 訊號強度
  • 若問題持續，請聯繫總部技術支援

👥 會員與訂單問題

Q8: 顧客要求退款怎麼處理？

A:

  1. 確認顧客的退款原因
  2. 檢查訂單狀態（是否符合退款條件）
  3. 至「訂單管理」→「申請退款」
  4. 填寫退款原因與金額
  5. 提交後由 ONE桌遊總部審核
  6. 審核通過後，款項自動退回顧客

Q9: 顧客反映收不到驗證碼簡訊？

A:

  • 請顧客確認手機號碼是否正確
  • 檢查手機是否有訊號
  • 某些電信業者可能延遲，請稍候 1-2 分鐘
  • 建議顧客重新發送驗證碼

Q10: 如何幫會員修改訂單時間？

A:

  1. 進入「訂單管理」
  2. 搜尋該會員的訂單
  3. 點擊「延長時間」或「提早結束」
  4. 系統會自動計算價差並請會員付款
